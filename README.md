# Digital Twin of Slovenia

This repository contains the code and resources for the Digital Twin of Slovenia project.

## Sentinel Hub Custom Scripts

A collection of custom scripts for Sentinel Hub, which can be fed to the visualization on Copernicus Space Data Ecosystem via the URL is in the `custom scripts` folder.
These scripts are designed to enhance the visualization and analysis of satellite imagery for various applications.

The scripts are adopted from the [Sentinel Hub Custom Scripts repository](https://custom-scripts.sentinel-hub.com/custom-scripts/).
